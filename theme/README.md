# Zellig Care WordPress Theme

A WordPress theme conversion of the Zellig Care mental health services website.

## Installation

1. Upload the `theme` folder to your WordPress `wp-content/themes/` directory
2. Navigate to Appearance > Themes in your WordPress admin
3. Activate the "Zellig Care" theme

## Features

- Responsive design
- Custom header and footer
- WordPress menu support
- Widget areas
- Custom post type support
- Accessibility features

## Theme Structure

```
theme/
├── css/                    # CSS files
├── js/                     # JavaScript files
├── styles/                 # Additional styles
├── inc/                    # Theme includes
├── header.php              # Header template
├── footer.php              # Footer template
├── index.php               # Main template
├── page.php                # Page template
├── functions.php           # Theme functions
├── style.css               # Main stylesheet
└── README.md               # This file
```

## Customization

The theme includes all original CSS and JavaScript files from the original HTML site. You can customize colors, fonts, and layout by modifying the CSS files in the `css/` and `styles/` directories.
